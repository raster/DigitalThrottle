# DigitalThrottle

An interface for the Leadrise Motor Controller which allows the use of 0-5V throttles instead of requiring (more expensive) 5K throttles.

We use a Microchip Technologies MCP4131-104E/P Digital Potentiometer controlled by an Arduino to emulate the resistive throttle.
